Linux/Unix 下的路径:
路径: path
    是用来记录一个文件或文件夹位置的字符串
    如:
     /home/tarena
     /home/tarena/aid1808/linux/day01.txt

根(root) /

Linux路径分为两种：
    绝对路径
    相对路径

绝对路径:
    以'/'字符开头的路径为绝对路径
    (在任何时候，一个文件的绝对路径都是唯一的)

相对路径:
    不以＇／＇开头的路径为相对路径
    开始符号有：
        1) 文件/文件夹名称
        2) . 当前文件夹
        3) .. 上一级文件夹
        4) ~ 用户主目录（家目录)
        5) ~/aid1808/**

cd 命令
    作用：
        改变当前工作路径
    格式：
        cd [目录名]
    示例：
        cd /home
        cd /
        cd      #默认不写目录名相当于　cd　～
        cd ..   #去上一级目录
        cd -  　# 切换到进入这个文件夹之前的文件夹

rmdir 命令:
  作用:
    删除一个或多个文件夹(文件夹内必须为空)
  格式:
    rmdir [选项] 文件夹名
  常用选项

  示例:

touch命令：
    作用：创建文件或更新文件的修改时间
    如：
        touch a.txt

rm 命令：
    删除文件或文件夹
    格式：
        rm [ 选项]　文件或文件夹
    常用选项：
        －r 递归删除文件夹内部的文件夹和文件
        －i 删除前给出提示（y代表yes, n代表no）
        -f 强制删除，不给任何提示
