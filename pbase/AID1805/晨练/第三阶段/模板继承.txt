模板的继承
	1.在父模板中
		必须要标识出哪些内容在子模板中是允许被修改的
		标签:
		{% block 名称 %}
			父模板中的内容
		{% endblock %}
		block的作用:
		1.在父模板中是正常显示的
		2.在子模板中,如果不修改block中的内容的话则按照父模板中的内容进行显示.
        要是修改了的话,则按照子模板中的内容显示了.
	2.在子模板中
		1.指定继承自哪个模板
		{% extends '父模板名称' %}
		2.增加block标签,改写属于自己的内容
			{% block 名称 %}
			{% endblock %}
